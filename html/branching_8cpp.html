static

Branching execution method for fractional LP solutions. This method is called by SCIP when it is needs to perform branching In our implementation, this single function inspects the variables at the current node, decides on the best of the 3 branching rules to apply, and applies it. If branching options have been exhausted, it reports this to scip

Its definition must be exactly so to dialogue correctly with SCIP. We refer to SCIP's documentation for SCIP_DECL_BRANCHEXECLP
